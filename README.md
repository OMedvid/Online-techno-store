# 🛒 Online Store

This is a modern **e-commerce web application built with React and TypeScript**.

The app allows users to browse products, view detailed information, manage favorites, and interact with a fully functional shopping cart.

---

## 🚀 Features

- Modern component-based architecture
- Responsive layout for all screen sizes
- Sticky header with navigation, favorites, and cart
- Interactive home page with product sliders
- Product catalog with filtering, sorting, and pagination
- Product details pages
- Shopping cart with quantity control and total calculation
- Favorites system
- Search with debounce
- Data persistence using localStorage
- Error handling and NotFound page
- Skeleton loaders for improved UX

---

## 🛠 Technologies Used

- React
- TypeScript
- SCSS
- React Router
